# PXG Tools Legacy Static

Esta pasta guarda a versao estatica antiga do PXG Tools apenas como referencia.

O frontend oficial do projeto fica em `../frontend/`, usando Next.js, TypeScript e Tailwind. O deploy publico deve usar o `netlify.toml` da raiz, que aponta para `frontend/`.

Novas telas, correcoes e deploys devem ser feitos somente no frontend oficial.
